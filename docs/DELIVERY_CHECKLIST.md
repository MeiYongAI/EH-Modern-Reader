## 🔧 使用方法

### 1. 快速安装（2 步骤）

```bash
# 步骤 1: 进入扩展页面
chrome://extensions/    # Chrome
edge://extensions/      # Edge

# 步骤 2: 加载扩展
1. 开启"开发者模式"
2. 点击"加载已解压的扩展程序"
3. 选择 eh-reader-extension 文件夹
```

### 2. 生成图标（可选）

```bash
# 方法 A: 在浏览器打开
open icon-generator.html

# 方法 B: 临时禁用图标
编辑 manifest.json，注释掉 icons 配置
```

### 3. 开始使用

```
1. 访问 e-hentai.org
2. 打开任意画廊
3. 点击 MPV 按钮
4. 🎉 自动启用新阅读器
```
