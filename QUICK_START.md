# ⚡ 快速开始指南

> 5 分钟快速上手 EH Modern Reader

## 🎯 一分钟概览

EH Modern Reader 是一个现代化的 E-Hentai 阅读器浏览器扩展，自动替换原版 MPV 阅读器。

**核心特点：** 现代化 UI、深色模式、智能预加载、进度记忆

---

## 📦 安装（3 步骤）

### Chrome / Edge 用户

1. **下载项目**
   ```powershell
   # 如果你已经有项目文件夹，跳过此步
   ```

2. **打开扩展页面**
   - 在地址栏输入：`chrome://extensions/` 或 `edge://extensions/`
   - 开启右上角的 **"开发者模式"**

3. **加载扩展**
   - 点击 **"加载已解压的扩展程序"**
   - 选择 `eh-reader-extension` 文件夹
   - ✅ 完成！

### Firefox 用户

1. **打开调试页面**
   - 在地址栏输入：`about:debugging#/runtime/this-firefox`

2. **临时载入**
   - 点击 **"临时载入附加组件"**
   - 选择项目中的 `manifest.json` 文件
   - ✅ 完成！

## 🚀 使用方法（超简单）

### 第一次使用

1. 访问 [E-Hentai](https://e-hentai.org) 或 [ExHentai](https://exhentai.org)
2. 打开任意画廊详情页
3. 点击顶部的 **MPV** 按钮
4. 🎉 新阅读器自动启动！

### 基本操作

| 操作 | 方法 |
|------|------|
| **下一页** | 点击图片右侧 / 按 `→` 键 / 按 `空格` |
| **上一页** | 点击图片左侧 / 按 `←` 键 |
| **跳转** | 点击左侧缩略图 |
| **设置** | 点击顶部设置按钮 ⚙️ |
| **全屏** | 按 `F11` 或点击全屏按钮 |
| **深色模式** | 点击月亮图标 🌙 |

### 全部快捷键

```
← / A         上一页
→ / D / 空格   下一页
Home          第一页
End           最后一页
F             切换侧边栏
F11           全屏模式
Esc           退出全屏/关闭面板
```

## ⚙️ 常用设置

点击顶部 ⚙️ 按钮打开设置面板：

1. **图片适配模式**
   - 适应窗口（推荐）- 图片自动调整大小适应屏幕
   - 适应宽度 - 填满屏幕宽度
   - 适应高度 - 填满屏幕高度
   - 原始大小 - 显示图片原始尺寸

2. **图片对齐**
   - 居中（默认）
   - 左对齐
   - 右对齐

3. **预加载下一页** ✓ 推荐开启
   - 自动预加载下一页图片，翻页更流畅

4. **平滑滚动** ✓ 推荐开启
   - 缩略图列表平滑滚动效果

## ❓ 常见问题

### Q: 为什么图片加载很慢？
**A:** 当前版本使用缩略图演示。完整版需要调用 E-Hentai API 获取高清图片。

### Q: 进度没有保存？
**A:** 
- 检查浏览器是否允许 localStorage
- 隐私模式下可能无法保存
- 清除浏览器数据会丢失进度

### Q: ExHentai 无法使用？
**A:** 
- 确保已登录 ExHentai（需要会员账号）
- 检查 Cookie 是否有效
- 尝试在 E-Hentai 测试是否正常

### Q: 快捷键不工作？
**A:**
- 确保页面有焦点（点击页面任意位置）
- 输入框焦点时快捷键不响应（故意设计）
- 检查是否与浏览器快捷键冲突

### Q: 如何卸载扩展？
**A:**
- Chrome/Edge: 扩展管理页面点击"移除"
- Firefox: about:addons 点击"移除"
